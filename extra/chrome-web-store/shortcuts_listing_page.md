# Shortcuts’ listing page

## Product details

###### English description

``` plaintext
Want to open a new tab to the right with your keyboard? Shortcuts lets you bind actions to keyboard shortcuts by using the “Extension shortcuts” interface. Actions are accessible using text input, keyboard shortcuts or mouse clicks in the extension’s popup.

Shortcuts is a browser extension that lets you perform common tasks with your keyboard. Alt+Space (Ctrl+Space on Windows) is the main keyboard shortcut. Use it to open the extension’s popup—aka “Vim” mode. Press Slash to search commands and Escape to cancel. If you need explanations of the shortcuts—Right-click the Shortcuts toolbar button and select “Documentation”. The keyboard shortcuts are fully customizable. Navigate to chrome://extensions/shortcuts to configure global keyboard shortcuts. You can also configure the popup keys by importing and exporting settings in the “Options” page.

Shortcuts’ features are offered via subscription to GitHub Sponsors at https://github.com/sponsors/taupiqueur. Amount must be at least $1. One-time sponsorships are active for 30 days. When the extension is first installed, you have 60 minutes of full access to explore all features. Once the trial expires, you will need to verify your sponsorship to continue using Shortcuts. To verify your sponsorship—Right-click the Shortcuts toolbar button and select “Sponsor this project”. There, you’ll find instructions to become a sponsor and verify your sponsorship. Note: You need an internet connection and a GitHub account for verification.

See the homepage for more information: https://taupiqueur.github.io/chrome-shortcuts
```

Accédez à chrome://extensions/shortcuts pour configurer les raccourcis globaux. Vous pouvez également configurer les raccourcis du menu de l’extension en important et exportant les paramètres dans la page des options.

Les fonctionnalités de Shortcuts sont proposées via un abonnement GitHub Sponsors à l’adresse https://github.com/sponsors/taupiqueur. Le montant doit être d’au moins 1 $. Les paiements ponctuels sont valables pendant 30 jours. Une fois l’extension installée, vous disposez de 60 minutes d’accès complet pour découvrir l’ensemble des fonctionnalités. Une fois la période d’essai terminée, vous devrez vérifier votre statut de sponsor pour continuer à utiliser Shortcuts. Pour vérifier votre statut de sponsor—Faites un clic droit sur l’icône de l’extension et sélectionnez « Devenir sponsor de taupiqueur ». Vous y trouverez les instructions pour devenir sponsor et vérifier votre statut. Remarque : Vous devez disposer d’une connexion internet et d’un compte GitHub pour vérifier votre statut.
