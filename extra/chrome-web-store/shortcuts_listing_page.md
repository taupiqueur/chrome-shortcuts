# Shortcuts’ listing page

## Product details

###### English description

``` plaintext
Want to open a new tab to the right with your keyboard? Shortcuts lets you bind actions to keyboard shortcuts by using the “Extension shortcuts” interface. Actions are accessible using text input, keyboard shortcuts or mouse clicks in the extension’s popup.

Shortcuts is a browser extension that lets you perform common tasks with your keyboard.

Alt+Space (Ctrl+Space on Windows) is the main keyboard shortcut. Use it to open the extension’s popup—aka “Vim” mode. Press Slash to search commands and Escape to cancel.

If you need explanations of the shortcuts—Right-click the Shortcuts toolbar button and select “Documentation”.

The keyboard shortcuts are fully customizable.

Navigate to chrome://extensions/shortcuts to configure global keyboard shortcuts. You can also configure the popup keys by importing and exporting settings in the “Options” page.

See the homepage for more information: https://taupiqueur.github.io/chrome-shortcuts
```

###### Description française

``` plaintext
Vous souhaitez ouvrir un nouvel onglet à droite au clavier ? Shortcuts vous permet d’assigner des actions à des raccourcis dans les options « Raccourcis clavier ». Les actions sont accessibles via la saisie de texte, les raccourcis clavier ou la souris dans le menu de l’extension.

Shortcuts est une extension Chrome qui vous permet d’effectuer des tâches rapidement au clavier.

Alt+Espace (Ctrl+Espace sous Windows) est le raccourci principal. Utilisez-le pour ouvrir le menu de l’extension—aka mode « Vim ». Appuyez sur Ctrl+F pour rechercher des commandes ou Échap pour quitter.

Si vous avez besoin d’explications sur les raccourcis—Faites un clic droit sur l’icône de l’extension et sélectionnez « Documentation ».

Les raccourcis clavier sont entièrement personnalisables.

Accédez à chrome://extensions/shortcuts pour configurer les raccourcis globaux. Vous pouvez également configurer les raccourcis du menu de l’extension en important et exportant les paramètres dans la page des options.

Pour en savoir plus, consultez la page d’accueil : https://taupiqueur.github.io/chrome-shortcuts/fr/
```
