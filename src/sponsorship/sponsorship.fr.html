Page de sponsoring Shortcuts

Devenez sponsor de taupiqueur, puis vérifiez votre statut.

Votre sponsoring est nécessaire pour utiliser Shortcuts.

Options de sponsoring
  • Devenir sponsor de taupiqueur
  • Accès complet à l’ensemble des fonctionnalités. Aucune carte de crédit n’est nécessaire.

Devenir sponsor de taupiqueur

Devenez sponsor de taupiqueur à l’adresse https://github.com/sponsors/taupiqueur. Une fois connecté, sélectionnez l’un des paliers disponibles ou choisissez un montant personnalisé. Si vous êtes nouveau sur GitHub, vous devrez créer un compte.

Le montant doit être d’au moins 1 $. Les paiements ponctuels sont valables pendant 30 jours.

  • Se connecter avec GitHub pour vérifier votre sponsoring

Se connecter avec GitHub pour vérifier votre sponsoring

  1. Accédez à https://github.com/login/device et sélectionnez le compte que vous souhaitez utiliser.
  2. Saisissez votre code à usage unique : 0000-0000.
  3. Cliquez sur « Continuer ». GitHub devrait afficher une page vous demandant d’autoriser l’app taupiqueur à vérifier votre identité GitHub et à connaître les ressources auxquelles vous avez accès.
  4. Cliquez sur « Autoriser l’app taupiqueur ». Après ceci, Shortcuts devrait être capable de vérifier votre statut de sponsor.
