Shortcuts utilise le Chrome DevTools Protocol pour effectuer des captures d’écran de l’ensemble de la page. Lorsque ce protocole est utilisé, Chrome affiche le message : « Shortcuts a démarré le débogage de ce navigateur » avec un bouton « Annuler ». Vous pouvez attendre la fin de l’opération ou cliquer sur « Annuler » pour fermer la session de débogage et interrompre la capture.
